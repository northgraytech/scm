# SinCityMafia (SCM) Website

Modern, responsive landing experience for the SinCityMafia creative collective. Built with Next.js, React, and handcrafted animations to highlight the crew, their music, and visuals.

## Getting Started

```bash
npm install
npm run dev
```

Visit `http://localhost:3000` to explore the experience locally.

## Available Scripts

- `npm run dev` – Starts the development server with hot reload.
- `npm run build` – Creates a production build of the site.
- `npm run start` – Serves the production build.

## Tech Stack

- **Next.js 14** for the application framework.
- **React 18** for component-driven UI.
- **CSS** for the custom cinematic styling, parallax background, and animations.

## Features

- Cinematic landing hero with SCM branding and smooth entrance animations.
- About, Members, Music, Gallery, and Contact sections tailored to the collective.
- Interactive member grid with hover bios and social links.
- Embedded music players and teaser card.
- Responsive gallery grid and stylized contact form with social icons.
